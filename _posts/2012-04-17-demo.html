Install and launch apps in Registry app.

This is kitchen sink sample. It demonstrates all UI components and themes and dashboard. You can launch demo in webkit based browser (Chrome, Safari, Mobile Safari, Android Browser)

Launch in a new tab
Tip1:   Long click the bottom area of left iPhone screen!
Tip2:   Scan the QR code with your smartphone!
Tip3:   Navigate to following URL in your smartphone browser. http://mobello.github.com/src/launcher.html

Setting

You can change a current theme and font with setting app.

Regitstry

It shows registered apps in app registry. You can search and install an app on the dashboard.

UI Demo

It demonstrates all UI Components in Mobello. Run UI Demo (in single app)
