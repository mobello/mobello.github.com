Class tau.ui.Dialog

    tau.ui.ActionSheet

팝업형태의 다이얼로그 컴포넌트이다.

tau.ui.ModalSceneController과 차이점은 Dialog 컴포넌트는 간단하게 UI를 구성하고자 할 때, 복잡한 이벤트 처리가 없는 경우에 유용하게 사용된다.

대표적인 기능 :

  • #open : Dialog 컴포넌트를 화면에 보여준다.
  • #close : Dialog 컴포넌트를 화면에서 사라지게 한다.

대표적인 이벤트 :

#event:tau.ui.Dialog.EVENT_CLOSE
Dialog 컴포넌트의 close 버튼을 탭하거나, mask영역을 탭해서 Dialog가 화면에서 사라지면 해당 이벤트가 발생한다.
