Appresto Kitchen Sink

 

Appresto Kitchen Sink는 Appresto에서 제공되는 모든 UI component와 Theme, 그리고 Dashboard기능을 보여줍니다.
이 데모는 WebKit 기반의 브라우저(Chrome, Safari)와 스마트폰 및 타블릿에서 확인해 보실 수 있습니다.

팁1: 왼쪽 아이폰의 바탕화면 하단을 0.5초간 눌러보세요! [새창으로 보기]
팁2: 스마트폰 및 타블릿에서는 우측의 QR코드를 찍어보세요!
팁3: http://appresto.github.com/src/launcher.html (아이폰 직접접속)

  • Setting

    • 전체 App의 Theme를 변경할 수 있습니다. 현재 Appresto UI Framework에서는
      • Black & white 톤의 Default theme
      • 그라데이션 없는 black & white 톤의 Simple theme
      • 아이폰과 흡사한 ios theme
      를 제공합니다.
    • 전체 App의 font를 변경할 수 있습니다. 현재 Appresto UI Framework에서는
      • Default font인 MuseoSans font
      • KT의 olleh font
      • Helvetica font
      를 제공합니다.


  • Registry

    • App Registry에 등록된 앱을 검색하고 Dashboard(바탕화면)에 설치할 수 있습니다.


  • UI 데모

    • Appresto에서 제공되는 모든 UI Component의 기능을 체험해 볼 수 있습니다. Appresto에서 제공되는 UI Component에는 
      • Label
      • Button, Checkbox, Radio
      • Picker, DatePicker
      • TextField, TextArea
      • Table, TableCell
      • Panel, ScrollPanel
      • Slider, Switch, Select, SegmentedButton, Spinner
      • ActivityIndicator
      • ToolBar
      • SystemDialog, Dialog, ActionSheet
      • ImageView,TextView
      • Carousel
      등이 있습니다.
