전체적인 Flow

외부 데이터를 가져와서 Table 컴포넌트에 동적으로 셀을 추가하는 것은 일반적으로 많이 사용하는 패턴입니다.

이 단락에서는 외부에서 데이터를 가져오는 부분, 셀을 구성하는 부분, 셀을 테이블에 추가하는 부분에 대해 대략적으로 그림으로 단계적으로 설명하겠습니다.

이 부분에 대한 설명이 잘 이해가 안된다면 걱정하지 마시고 Skip 하셔도 상관없습니다. 다음 단락에서 예제를 통해 다시 한번 설명하겠습니다.

 

  1. loadModel : 직접 tau.ui.Table#loadModel을 호출하거나, 더 보기 셀을 탭했을 때 해당 함수가 호출됩니다. 이 때 테이블에서는 tau.ui.Table.EVENT_MODEL_LOAD 이벤트를 발생 시킵니다.
  2. handleModelLoad: tau.ui.Table.EVENT_MODEL_LOAD 이벤트에 대한 콜백함수입니다.
    1. data load : 데이터를 가져오는 부분을 사용자가 처리합니다.
    2. addNumOfCells : 데이터를 가져왔다면 해당 데이터의 개수를 설정합니다. 그럼 이때 테이블에서는 tau.ui.Table.EVENT_CELL_LOAD 이벤트를 발생 시킵니다.
  3. handleCellLoad: tau.ui.Table.EVENT_CELL_LOAD이벤트에 대한 콜백함수입니다.
    1. cell create : 셀을 생성합니다.
    2. add : 생성된 셀을 테이블에 add 합니다.

 

외부 데이터 가져오기

이벤트 : tau.ui.Table.MODEL_LOAD

설명 : tau.ui.Table#loadModel 함수를 직접 호출하거나, 더 보기 셀을 탭해서 내부적으로 호출할 때 tau.ui.Table.EVENT_MODEL_LOAD 이벤트를 발생시킵니다.

payload에 대한 설명

  • start : 시작 인덱스값
  • size: 가져올 데이터 개수. Table 생성자자 옵션에서 listSize값.
  • page: 페이지네이션을 사용할 경우 가져올 페이지 번호

 

tau.ui.Table.MODEL_LOAD 이벤트에 대한 콜백함수를 지정합니다.

table.onEvent(tau.ui.Table.EVENT_MODEL_LOAD, this.handleModelLoad, this);

 

tau.ui.Table.EVENT_MODEL_LOAD 이벤트에 대한 콜백함수에서는 payload에 있는 정보를 가지고 내부 혹은 외부(서버)를 통해 데이터를 가져 옵니다.

외부에서 데이터를 가져오는 부분은 이 예제에서 사용하는 Google Feed API를 참고하시기 바랍니다.

여기에서는 외부에서 데이터를 가져오는 부분에 대한 것은 따로 설명하지 않겠습니다.

 

handleModelLoad: function (e, payload) {
... // 데이터 가져오기 로직
}

 

데이터를 가져온 후에 Table 컴포넌트 객체에 tau.ui.Table#addNumOfCells 함수에 데이터 크기를 설정합니다.

handleModelLoad: function (e, payload) {
  var table = e.getSource();
  ...
  table.addNumOfCells(데이터크기); 
}

셀 생성하기

데이터를 로드가 완료되면 데이터 개수만큼 셀을 생성해야 합니다. 

addNumOfCells 함수에 데이터 크기를 설정하게 되면 설정한 개수만큼 루프를 돌면서 tau.ui.Table.EVENT_CELL_LOAD 이벤트를 발생시킵니다.

table.onEvent(tau.ui.Table.EVENT_CELL_LOAD, this.handleCellLoad, this);

이벤트 : tau.ui.Table.EVENT_CELL_LOAD

설명 : tau.ui.Table#addNumOfCells 함수를 호출하게되면 tau.ui.Table.EVENT_CELL_LOAD 이벤트를 발생시킵니다.

payload에 대한 설명

  • index : 로드한 데이터에 접근하기 위한 인덱스값
  • offset : 로드한 데이터에 접근하기 위한 오프셋값
  • 현제 셀이 로드한 데이터에 접근하기 위해서는 index + offset 으로 접근할 수 있습니다.
handleCellLoad: function (e, payload) {
  var table = e.getSource();
  var index = payload.index;
  var offset = payload.offset;
  var data = this.data[index + offset];
  var tableCell = new tau.ui.TableCell({title: data.title, subTitle: data.subtitle});
  
  ....
}

 

동적으로 셀 추가하기

셀이 생성이 완료되면 테이블에 추가합니다.

handleCellLoad: function (e, payload) {
  ....
  table.add(tableCell);
}

addNumOfCells 함수에 설정한 크기만큼 셀이 추가가 완료되면 자동으로 Scene(화면)에 추가된 셀이 반영됩니다.
